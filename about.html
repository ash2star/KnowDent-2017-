Introduction

Inspite of widespread awareness, dental health is still neglected, it is one of the most faced problem by people after common cold.

KnowDent is an attempt to increase the knowledge about the importance of dental health, thereby displaying possible dental problems with respect to the symptoms.



How KnowDent works?

i) If you are a new User please signup or if you an existing user kindly login into the system.

ii) Once the login page loads it asks for the input (symptoms) from the user.

iii) After accepting the input it displays information regarding the set of possible dental issues and some preventive measures.



Why KnowDent?

1.Less Cost
2.Faster detection
3.Saves time
4.Misperception of people
5.Less travel
6.Reduces anxiety and fear
7.No appointment needed
